# 📚 AI-Based Book Genre Classification

This project develops an AI-based system that automatically classifies book genres using Machine Learning and Natural Language Processing (NLP). The model analyzes book descriptions and predicts their most likely genre.

---

## 🎯 Project Objective
The goal of this project is to build a machine learning model capable of predicting book genres from textual descriptions, helping organize digital libraries and improve recommendation systems.

---

## 🧠 Technologies Used
- Python
- Natural Language Processing (NLP)
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## ⚙️ Machine Learning Models
The following models were trained and evaluated:

- Logistic Regression  
- Support Vector Machine (SVM)  
- Naive Bayes  
- Random Forest  

After hyperparameter tuning using "GridSearchCV", the "Support Vector Classifier (SVC)" achieved the best performance.

---

## 📊 Results
| Model | Accuracy |
|------|---------|
| Logistic Regression | 63.9% |
| SVM (Best Model) | 65.5% |
| Random Forest | 44.9% |
| Naive Bayes | 40.1% |

The SVM model achieved the highest accuracy of 65.5%, successfully meeting the project requirement.
